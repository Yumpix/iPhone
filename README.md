# Учет телефонов

Простое веб-приложение для учета закупок и продаж телефонов.

## Что умеет

- Учет телефонов (закупка и продажа).
- Автоматические партии по воскресеньям.
- Автоматический расчет прибыли.
- Дашборд за текущий месяц.
- Таблица телефонов и отметка о продаже.

## Стек

- Next.js
- TypeScript
- Tailwind CSS
- Prisma
- SQLite

## Как запустить локально (самый простой путь)

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте базу и таблицы:
   ```bash
   npx prisma migrate dev --name init
   ```
3. Запустите приложение:
   ```bash
   npm run dev
   ```
4. Откройте [http://localhost:3000](http://localhost:3000).

## Как пользоваться

1. В форме «Новая закупка» укажите модель и цену закупки.
2. Приложение автоматически определит партию по ближайшему воскресенью.
3. В таблице укажите цену продажи и нажмите «Отметить проданным».
4. Дашборд обновится и покажет прибыль за текущий месяц.
